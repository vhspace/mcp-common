# gpu-diag-mcp

MCP server for GPU XID/SXid diagnostics, NCCL test analysis, and ECC monitoring

## CLI: `gpu-diag-cli`

Run `gpu-diag-cli --help` for all commands.
Install: `uvx --from git+https://github.com/vhspace/gpu-diag-mcp@v1.1.1 gpu-diag-cli`

## MCP Server

```bash
uvx --from git+https://github.com/vhspace/gpu-diag-mcp@v1.1.1 gpu-diag-mcp
```
