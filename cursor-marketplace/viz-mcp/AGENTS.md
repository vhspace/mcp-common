# viz-mcp

MCP server and CLI for data visualization -- render shareable reports, charts, and tables

## CLI: `viz-cli`

Run `viz-cli --help` for all commands.
Install: `uvx --from git+https://github.com/vhspace/viz-mcp@v1.1.1 viz-cli`

## MCP Server

```bash
uvx --from git+https://github.com/vhspace/viz-mcp@v1.1.1 viz-mcp
```
