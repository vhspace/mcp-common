# weka-mcp

MCP server for Weka storage system management and monitoring

## CLI: `weka-cli`

Run `weka-cli --help` for all commands.
Install: `uvx --from git+https://github.com/vhspace/weka-mcp@v0.6.0 weka-cli`

## MCP Server

```bash
uvx --from git+https://github.com/vhspace/weka-mcp@v0.6.0 weka-mcp
```

### Required env vars

- `WEKA_HOST`: ${WEKA_HOST}
- `WEKA_PASSWORD`: ${WEKA_PASSWORD}
